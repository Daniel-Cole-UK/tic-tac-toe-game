Free Code Camp Front End Development Projects - Build a Tic Tac Toe Game

Task: Create a web page containing an Tic Tac Toe game where the user can play against the computer.

User Stories:

- As a user, I can play a game of Tic Tac Toe with the computer.
- My game will reset as soon as it's over so I can play again.
- As a user, I can choose whether I want to play as X or O.

Created Using:

- HTML5
- CSS3
- jQuery
- Animate.css
- Google fonts
